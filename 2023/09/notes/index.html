Notes

Loop: When you want to do the same action with every item in a list, you can use Python’s for loop. (p. 50)

Expected an indented block after ‘for’ statement on line 3

Every indented line following the line for magician in magicians is considered inside the loop, and each indented line is executed once for each value in the list. (p. 52)

Any lines of code after the for loop that are not indented are executed once without repetition. (p. 52)

The colon at the end of a for statement tells Python to interpret the next line as the start of a loop.

Attention to indentation:

(1) Always indent the line after the for statement in a loop. (inclusing the first line and additional lines, if you want to see a certain action repeated once for each item in the list.) (p. 53)

(2) If you accidentally indent a line that doesn’t need to be indented, Python informs you about the unexpected indent: IndentationError: unexpected indent. (p. 54)

要想给一个 list 里的 item 发 message，如果直接会某个 item 直接发的话，要在 print() 里写成 list[X] 的形式，如果要给所有 item 发的话，可以用 ‘for’ loop，在 ‘for’ statement 后面的 print() 里只要写成那个 valuable 的格式就可以。（参考 bicycles.py 和 magicians.py）

Logic errors related to indentation:

(1) If you expect to see a certain action repeated once for each item in a list and it’s executed only once, determine whether you need to simply indent a line or a group of lines.

(2) If an action is repeated many times when it should be executed only once, you probably need to unindent the code for that action.
